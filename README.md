# Fine
Refined long reverberation effect,  
can use as `REVERB EFFECTS`.

# Parameters
- shape (NTS-1 mkII: A)  
Reverberation time.
- alt (NTS-1 mkII: B)  
Send level.
- shift depth  
dry/wet mixing ratio.

# How to build
1. Clone (or download) this project.
1. Copy to under `platform/nts-1_mkii`.
1. See README.md under `platform/nts-1_mkii`.
